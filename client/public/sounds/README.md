# VINTAGE COMPUTING SOUND EFFECTS

This directory contains AUTHENTIC MAINFRAME ERA sound effects for the NECRO-BANK system.

## Sound Files Required

### 1. teletype.mp3
- **Description**: Mechanical teletype clacking sound
- **Usage**: Played when terminal text appears
- **Authentic Source**: ASR-33 Teletype or IBM 2741 Terminal
- **Duration**: ~0.5-1 second
- **Recommended**: Short burst of mechanical typing

### 2. tapedrive.mp3
- **Description**: Magnetic tape drive motor sound
- **Usage**: Played during calculation execution
- **Authentic Source**: IBM 729 Tape Drive
- **Duration**: ~2-3 seconds
- **Recommended**: Motor startup, running, and stop sequence

### 3. cardreader.mp3
- **Description**: Punch card reader mechanical sound
- **Usage**: Played when form is submitted
- **Authentic Source**: IBM 029 Card Punch or IBM 2540 Card Reader
- **Duration**: ~1-2 seconds
- **Recommended**: Card feeding and reading mechanism

### 4. bell.mp3
- **Description**: Console bell or chime
- **Usage**: Played on successful calculation completion
- **Authentic Source**: IBM Console Bell or Teletype Bell
- **Duration**: ~0.5 seconds
- **Recommended**: Single clear bell tone

### 5. buzzer.mp3
- **Description**: Error buzzer or alarm
- **Usage**: Played when errors occur
- **Authentic Source**: IBM Console Buzzer or Error Alarm
- **Duration**: ~0.5-1 second
- **Recommended**: Harsh buzzer or alarm tone

## File Format Requirements

- **Format**: MP3 (for broad browser compatibility)
- **Bitrate**: 128-192 kbps
- **Sample Rate**: 44.1 kHz
- **Channels**: Mono or Stereo
- **Size**: Keep under 100KB per file for fast loading

## Testing

Test all sounds at various volume levels to ensure:
- No clipping or distortion
- Appropriate duration (not too long)
- Clear and recognizable
- Authentic vintage character
