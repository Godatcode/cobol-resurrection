# CLI Tools

This directory contains command-line interface tools for the Necro-Bridge toolkit.

## Planned Commands

- `necro-bridge` - Main CLI entry point
- `init.js` - Initialize bridge for a legacy language
- `detect.js` - Auto-detect legacy binaries in project
- `serve.js` - Start bridge server
- `test.js` - Test legacy binary integration

The CLI provides a developer-friendly interface for setting up and managing legacy bridges.
