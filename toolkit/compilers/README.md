# Compiler Utilities

This directory contains wrapper utilities for legacy language compilers.

## Planned Utilities

- `cobol-compiler.js` - GnuCOBOL compilation wrapper
- `fortran-compiler.js` - gfortran compilation wrapper
- `pascal-compiler.js` - Free Pascal compilation wrapper
- `basic-compiler.js` - FreeBASIC compilation wrapper
- `multi-compiler.js` - Universal compiler detection and execution

These utilities provide a consistent interface for compiling legacy code across different languages.
