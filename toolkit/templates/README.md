# Bridge Templates

This directory contains pre-built bridge server templates for various legacy languages.

## Planned Templates

- `cobol-bridge.js` - Bridge template for COBOL binaries
- `fortran-bridge.js` - Bridge template for FORTRAN binaries
- `pascal-bridge.js` - Bridge template for PASCAL binaries
- `basic-bridge.js` - Bridge template for BASIC binaries

Each template provides a standardized Express.js API wrapper for the legacy binary.
