# PASCAL Code Generation Template (1970 Era)

## Syntax Rules
- Use Standard Pascal (ISO 7185)
- Program structure: PROGRAM → VAR → BEGIN → END.
- All statements end with semicolon (except before END)
- Case-insensitive keywords (use UPPERCASE for clarity)
- Strong typing required
- Block structure with BEGIN/END
- Comments in { } or (* *)

## Standard Structure
```pascal
PROGRAM ProgramName;
{ Generated by NECRO-BRIDGE }

VAR
    variable1: REAL;
    variable2: INTEGER;
    result: REAL;

BEGIN
    { Read input }
    ReadLn(variable1);
    ReadLn(variable2);
    
    { Perform calculation }
    result := variable1 * variable2;
    
    { Output result }
    WriteLn('RESULT: ', result:0:2);
END.
```

## Example: Tax Calculator
```pascal
PROGRAM TaxCalculator;
{ Progressive tax bracket calculator }
{ Demonstrates structured programming }

VAR
    income: REAL;
    tax: REAL;
    bracket1, bracket2, bracket3: REAL;

BEGIN
    { Initialize tax brackets }
    bracket1 := 10000.0;
    bracket2 := 50000.0;
    bracket3 := 100000.0;
    
    { Read income from command line }
    IF ParamCount >= 1 THEN
        Val(ParamStr(1), income)
    ELSE
        income := 0.0;
    
    { Calculate progressive tax }
    tax := 0.0;
    
    IF income <= bracket1 THEN
        tax := income * 0.10
    ELSE IF income <= bracket2 THEN
        tax := bracket1 * 0.10 + (income - bracket1) * 0.20
    ELSE IF income <= bracket3 THEN
        tax := bracket1 * 0.10 + (bracket2 - bracket1) * 0.20 + 
               (income - bracket2) * 0.30
    ELSE
        tax := bracket1 * 0.10 + (bracket2 - bracket1) * 0.20 + 
               (bracket3 - bracket2) * 0.30 + (income - bracket3) * 0.40;
    
    { Output result }
    WriteLn('RESULT: ', tax:0:2);
END.
```

## Data Types
```pascal
VAR
    intVar: INTEGER;
    realVar: REAL;
    charVar: CHAR;
    boolVar: BOOLEAN;
    stringVar: STRING[80];
```

## Control Structures
```pascal
{ IF-THEN-ELSE }
IF condition THEN
    statement1
ELSE
    statement2;

{ CASE statement }
CASE variable OF
    1: statement1;
    2: statement2;
    ELSE statement3;
END;

{ FOR loop }
FOR i := 1 TO 100 DO
    statement;

{ WHILE loop }
WHILE condition DO
    statement;
```

## Procedures and Functions
```pascal
FUNCTION Calculate(x, y: REAL): REAL;
BEGIN
    Calculate := x * y;
END;

PROCEDURE DisplayResult(value: REAL);
BEGIN
    WriteLn('RESULT: ', value:0:2);
END;
```

## Error Handling Pattern
```pascal
IF income < 0.0 THEN
BEGIN
    WriteLn('ERROR: INVALID INCOME');
    Halt(1);
END;
```

## Output Format
- Always output in format: "RESULT: XXXX.XX"
- Use WriteLn with format specifier :0:2 for 2 decimal places
- Example: WriteLn('RESULT: ', value:0:2);
