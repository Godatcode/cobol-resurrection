# FORTRAN Code Generation Template (1957 Era)

## Syntax Rules
- Use FORTRAN IV fixed-format
- Column 1: Comment indicator (C or *)
- Columns 1-5: Statement labels (optional)
- Column 6: Continuation indicator (any character except 0 or space)
- Columns 7-72: FORTRAN statements
- Columns 73-80: Sequence numbers (ignored)
- All keywords in UPPERCASE
- Variable names: 6 characters max, start with letter
- IMPLICIT typing: I-N = INTEGER, A-H,O-Z = REAL

## Standard Structure
```fortran
C     PROGRAM NAME
C     GENERATED BY NECRO-BRIDGE
C     
      PROGRAM PROGNAME
      IMPLICIT NONE
      REAL VAR1, VAR2, RESULT
      INTEGER IVAR
      
C     READ INPUT FROM COMMAND LINE
      CALL GETARG(1, VAR1)
      CALL GETARG(2, VAR2)
      
C     PERFORM CALCULATION
      RESULT = VAR1 * VAR2
      
C     OUTPUT RESULT
      WRITE(*,100) RESULT
  100 FORMAT('RESULT: ', F10.2)
      
      STOP
      END
```

## Example: Trajectory Calculator
```fortran
C     BALLISTIC TRAJECTORY CALCULATOR
C     CALCULATES RANGE USING PHYSICS FORMULA
C     
      PROGRAM TRAJECTORY
      IMPLICIT NONE
      REAL VELOCITY, ANGLE, RANGE
      REAL PI, GRAVITY, RADIANS
      CHARACTER*20 ARG1, ARG2
      
C     CONSTANTS
      PI = 3.14159265
      GRAVITY = 9.81
      
C     READ COMMAND LINE ARGUMENTS
      CALL GETARG(1, ARG1)
      CALL GETARG(2, ARG2)
      READ(ARG1, *) VELOCITY
      READ(ARG2, *) ANGLE
      
C     CONVERT ANGLE TO RADIANS
      RADIANS = ANGLE * PI / 180.0
      
C     CALCULATE RANGE: R = (V^2 * SIN(2*THETA)) / G
      RANGE = (VELOCITY ** 2 * SIN(2.0 * RADIANS)) / GRAVITY
      
C     OUTPUT RESULT
      WRITE(*,100) RANGE
  100 FORMAT('RESULT: ', F10.2)
      
      STOP
      END
```

## Array Declaration
```fortran
      DIMENSION ARRAY(100)
      REAL MATRIX(10,10)
```

## Control Structures
```fortran
C     IF-THEN-ELSE
      IF (VAR .GT. 0.0) THEN
          RESULT = VAR * 2.0
      ELSE
          RESULT = 0.0
      ENDIF
      
C     DO LOOP
      DO 10 I = 1, 100
          SUM = SUM + I
   10 CONTINUE
```

## Error Handling Pattern
```fortran
      IF (VELOCITY .LE. 0.0) THEN
          WRITE(*,*) 'ERROR: INVALID VELOCITY'
          STOP 1
      ENDIF
```

## Output Format
- Always output in format: "RESULT: XXXX.XX"
- Use FORMAT statement with F10.2 for floating point
- WRITE(*,label) for output

```bash
gfortran -o legacy/fortran/{filename} legacy/fortran/{filename}.f
```
